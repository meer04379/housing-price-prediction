# Housing Price Prediction 🏠

This project predicts house prices using Machine Learning techniques. 
It analyzes different features such as location, area, number of rooms, and other factors to estimate the price of a house. 
The model is trained on a housing dataset and uses regression algorithms to make accurate predictions.

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Google Colab

## 📊 Model
- Linear Regression

## 📁 Dataset
- Housing dataset (CSV format)

## ▶️ How to Run
1. Open the notebook in Google Colab
2. Install required libraries (if needed)
3. Run all cells step by step
